遊戲說明：
	使用滑鼠左鍵點選button選取遊戲項目，
	進入遊戲歌曲選單可以使用滑鼠滾輪上下滑動選擇歌曲，
	點選歌曲名稱的button進入遊戲畫面，
	進入遊戲使用F、J鍵打擊鼓面(紅色)
		使用D、K鍵打擊鼓邊(藍色)

製作音符譜：
	在Home頁面點選Compose按鈕
